ML Project 3 Group 3

Command-line instructions 

First, compile: javac ID3.java

Then, run with: java ID3 inputDataFileName outputFileName

For example, to run with the test file task2.txt, use the command: java ID3 task2.txt output2.txt

Description of Files
ID3.java contains our implementation of the ID3 algorithm.
task2.txt, task3.txt, and task4.txt contain the training examples needed for Task 2, Task 3, and Task 4.
